# STM32WB55-RF-Layout-Antenna-Practice-Board
Learning-based STM32WB55CEUx PCB focused on RF layout considerations, antenna clearance, and impedance-aware routing
<img width="715" height="547" alt="RF Board" src="https://github.com/user-attachments/assets/4da19d62-c843-4275-b799-54a071dded6e" />
This learning-focused PCB project was designed using the STM32WB55CEUx microcontroller to better understand RF PCB design practices and wireless layout considerations. The component placement was planned based on practical layout understanding while considering RF constraints such as antenna clearance, grounding strategy, trace routing, and impedance-aware RF line design.

During this project, I explored important RF design concepts including noise reduction practices, minimizing interference near the antenna section, maintaining proper return paths, and following layout precautions for improved wireless performance and signal integrity.
